# Statistical-Analysis-and-Forecasting-for-Males-Mortality
This repository contains information and material for the Statistical Analysis and Forecasting for Males Mortality, a project for the course "Actuarial Modeling" (UNIPI, Department of Statistics and Insurance Science). The purpose of the project is to understand the mortality data, construct a mortality table and make forecasting with OLS and GLM. The project was written in R.
